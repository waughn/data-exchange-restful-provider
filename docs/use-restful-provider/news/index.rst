Sitecore Connect for News API 
===========================================================

`News API <https://newsapi.org>`_ is a simple and free HTTP REST API for searching and retrieving 
articles from various sources. The section covers steps to install and use, or build, a connector to 
read news articles from the service, and create Sitecore items for each article returned. 

.. note::
    The latest packages are available on `GitHub <https://github.com/waughn/data-exchange-restful-provider/releases>`_.
