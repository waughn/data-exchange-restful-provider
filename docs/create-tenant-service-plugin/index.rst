Create Tenant Web Service Plugins
===========================================================

This section walks you through the process of creating  
Data Exchange Framework Tenant Web Service Plugins for RESTful Provider 
and Sitecore Provider.

    The source code and the latest Sitecore packages are available on `GitHub <https://github.com/waughn/data-exchange-restful-provider>`_.

    See `Installation Guide for Data Exchange Tenant Web Service <https://dev.sitecore.net/Downloads/Data_Exchange_Framework/5x/Data_Exchange_Framework_500>`_
    and `Introducing the Tenant Service <https://doc.sitecore.com/developers/def/50/data-exchange-framework/en/the-tenant-service.html>`_
    for details about installing a tenant service and implementing a tenant service endpoint.
